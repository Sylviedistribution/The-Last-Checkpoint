## Application Security

One of the most valuable parts of the bootcamp was learning how to secure web applications.

Topics covered include:

* Implementing security best practices
* Cross-Site Scripting (XSS) prevention
* Cross-Site Request Forgery (CSRF) protection
* NoSQL Injection prevention
* HTTP Parameter Pollution (HPP)
* Protecting environment variables
* Securing HTTP headers with Helmet
* Rate Limiting and DDoS mitigation
* Input sanitization
* Secure cookies and session management

These practices taught me that building software is not only about features but also about protecting users and data.
